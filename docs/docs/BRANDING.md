# Erghi - Enterprise AI Chat Platform

**The Modern AI-Powered Chat Solution for Businesses**

> **Erghi** - Because conversations should flow naturally, powered by AI.

---

## Brand Identity

### Name
**Erghi** - Represents the seamless flow of conversations between businesses and customers, enhanced by AI intelligence.

### Tagline
"Intelligent Conversations, Flowing Naturally"

### Brand Values
- **Intelligent** - AI-powered responses that understand context
- **Seamless** - Effortless integration and user experience
- **Reliable** - Enterprise-grade infrastructure and security
- **Scalable** - Grows with your business needs
- **Open** - Multi-platform SDKs and extensible architecture

### Visual Identity
- **Primary Color**: `#3b82f6` (Blue) - Trust, technology, innovation
- **Secondary Color**: `#8b5cf6` (Purple) - Used in gradients alongside the primary blue (hero headings, CTAs, "Why Erghi" section)
- **Neutral**: `#0f172a` (Dark) / `#ffffff` (Light)

### Logo Concept
The shipped mark (see `ChatIcon.tsx` in erghi-landing, `icon.svg`/`logo.svg` across every app): a blue (`#3b82f6`) rounded square (8px corner radius on a 32x32 viewBox) containing a white speech-bubble shape with three small blue dots inside, suggesting an active conversation.

## Product Naming Convention

### Core Platform
- **Erghi Platform** - The complete system
- **Erghi Dashboard** - Admin interface
- **Erghi Widget** - Embeddable chat component
- **Erghi API** - RESTful API Gateway

### Services (Internal)
- **Identity Service** (formerly Auth Service)
- **Conversation Service** (formerly Chat Service)
- **Intelligence Service** (formerly AI Service)
- **Gateway Service** (formerly API Gateway)

### SDK Packages
- **@erghi/sdk** - JavaScript/TypeScript
- **erghi-sdk** - Python
- **@erghi/react** - React
- **erghi_sdk** - Flutter
- **ErghiSDK** - Swift
- **erghi-sdk** - Kotlin

### Database Schema
- **erghi_identity** - Identity Service DB
- **erghi_conversations** - Conversation Service DB
- **erghi_analytics** - Analytics Service DB
