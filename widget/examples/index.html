Erghi Widget Demo

This page embeds the Erghi chat widget via a single <script> tag. Use the buttons below to interact with the widget API. Point data-api-url at a running Erghi stack for a live conversation; without one the widget still renders and localizes.

Widget Embed Snippet

<script
  src="https://chat.erghi.ai/widget.js"
  data-widget-id="YOUR_WIDGET_ID"
  data-api-url="https://api.erghi.ai"
  data-direction="auto" <!-- 'auto' | 'ltr' | 'rtl' — auto mirrors for Arabic -->
  defer
></script>
