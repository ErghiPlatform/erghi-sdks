ChatFlow Widget Demo

This page embeds the ChatFlow chat widget via a single <script> tag. Use the buttons below to interact with the widget API and observe real-time events.

Widget Embed Snippet

<script
  src="http://localhost:5000/widget/chatflow.js"
  data-chatflow="YOUR_WORKSPACE_ID"
  data-api-url="http://localhost:5000"
  defer
></script>
